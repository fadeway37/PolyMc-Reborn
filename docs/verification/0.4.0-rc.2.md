# Verification record: PolyMc Reborn 0.4.0 RC2

This record covers `0.4.0-rc.2+26.1.2` in the independent
`fadeway37/PolyMc-Reborn` repository. Local results below were collected on
2026-07-20 from the fixed RC2 implementation tree, and were accepted only
after the command exit status and structured evidence agreed. The final
GitHub-hosted gates repeat the release-critical layers on the final commit.

Run and artifact identifiers are intentionally kept in the pull request,
draft release, and release audit rather than committed here: adding an
identifier produced for a commit would create a different candidate commit.

## Final hosted gates

The final commit must pass Standard CI, Production Client Playtest, Production
Multi-Client Playtest, the exact three-Mod matrix, Windows/Linux archive
reproducibility, Linux short and long soak, and the RC release gate in the
independent repository. Hosted evidence is accepted only after its repository,
head SHA, scenario totals, JUnit, Mod list, screenshots, process exits, cleanup,
artifact bytes, and attestation subjects have been inspected. A green workflow
badge alone is not evidence.

## Explicit gaps and metadata boundary

- The machine-wide `java` command is Dragonwell 21.0.6. Gradle resolved and
  used Eclipse Temurin 25.0.3+9 for compilation and Minecraft processes; no
  Java 21 compilation fallback was used.
- The isolated test client contains the bounded Fabric client driver and four
  permitted runtime components. A pure zero-Mod vanilla-client smoke remains
  `NOT_RUN` and is not a release claim.
- RC2 deliberately changes no `.java` file. Consequently, the Java constant
  used by in-game `about`, reports, and support-bundle metadata continues to
  identify RC1 even though Fabric, Gradle, Maven, SBOM, manifest, file names,
  and release metadata identify RC2. Changing that constant requires a
  separately authorized runtime-code change.
- Compatibility remains limited to the exact features and versions named by
  successful evidence; this record does not claim arbitrary Mod support.
