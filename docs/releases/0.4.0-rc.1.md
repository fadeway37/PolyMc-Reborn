# PolyMc Reborn 0.4.0 RC1

RC1 (`0.4.0-rc.1+26.1.2`) established the 0.4 stability candidate on Minecraft
26.1.2 and Java 25. It hardened session cleanup, registry sanitation, API
compatibility checks, three-mod external evidence, short/long soak orchestration,
reproducible release artifacts, and GitHub-hosted provenance without broadening
the project's unsafe automatic-conversion surface.

The immutable RC1 commit is
`f571ac5243518387c5b1a81c818dd8ffdd8db2de`; the annotated tag object is
`c5fb2c9afa1d46d397d3fdd3fd03ad769d9135ca`. The historical Draft Pre-release,
its 17 assets, workflow records, and original provenance remain in
[PolyMc-Reborn-Archive](https://github.com/fadeway37/PolyMc-Reborn-Archive).
They are retained as RC1 evidence and are not represented as evidence for RC2
in the independent repository.

Detailed feature notes remain in
[the original RC1 notes](../0.4-rc-release-notes.md), while the pre-RC baseline
audit remains in [baseline-audit-0.4-rc.md](../baseline-audit-0.4-rc.md).

RC1 did not automate a pure zero-mod vanilla client, enable creative reverse
mapping, enable packet fallback, infer arbitrary GUIs/entities, or promise
compatibility with every mod.
