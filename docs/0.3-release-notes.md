# PolyMc Reborn 0.3.0 Beta 1 release notes

Version `0.3.0-beta.1+26.1.2` targets exactly Minecraft 26.1.2, Fabric Loader
0.19.3, Java 25, Fabric API 0.155.2+26.1.2, and Polymer 0.16.5+26.1.2.

This Real-World Compatibility Beta adds a standalone API artifact, independent
consumer test Mod, explicit furnace properties, explicit entity passenger and
equipment composition, per-player pack policy, diagnostic policy/support
bundles, two-client production automation, two locked real-Mod scenarios, and
0.2 upgrade/mod-set preservation tests. Release artifacts include SBOM,
checksums, provenance, sources, Javadoc, and API signature.

It does not add arbitrary GUI/entity inference, broad packet rewriting,
creative reverse mapping, a required companion client, or trusted Modded
passthrough. Pure zero-Mod vanilla automation remains unimplemented. No claim
is made that every Mod works.

Back up the world and `config/polymc-reborn/`. The mapping schema remains `1`
and algorithm `reborn-2`; startup strictly validates the existing store. Review
the mapping diff and compatibility reports before players join.
